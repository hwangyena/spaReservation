# Itez Next Template for SSR(Server Side Rendering)

## 개발모드 실행

```
  npm run dev
```

## 빌드

```
  npm run build
```

## 빌드파일 실행

```
  npm start
```

## 포트 변경(9999번 예시)

### \*cross-env 방식

```
  PORT=9999 npm run dev
```

```
  PORT=9999 npm start
```

### \*npx 방식

```
  npx next dev -p 9999
```

```
  npx next start -p 9999
```
